# Blog Management System

A full-stack Blog Management System developed using Spring Boot, Spring Security, Thymeleaf, Hibernate, and MySQL.

## Features

- User Registration & Login
- Secure Authentication using Spring Security
- Password Encryption using BCrypt
- Create, Update and Delete Blog Posts
- View All Posts
- View Individual Post
- Comment System
- Category Management
- Search Posts
- User Profile
- Image Upload
- Author-based Access Control

## Tech Stack

- Java 17
- Spring Boot
- Spring Security
- Spring Data JPA
- Hibernate
- Thymeleaf
- MySQL
- HTML
- CSS
- Git
- GitHub

## Project Structure


## Installation

1. Clone the repository

```
git clone https://github.com/fruitless911nick/blog-management-system.git
```

2. Configure MySQL in `application.properties`

3. Run the project
